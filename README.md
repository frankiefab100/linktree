##  Installation Steps

1. Clone the repository

```bash
git clone https://github.com/frankiefab100/linktree.git
```

2. Change the working directory

```bash
cd linktree
```

3. Install dependencies

```bash
npm install
```

4. Run the app

```bash
npm run dev
```
